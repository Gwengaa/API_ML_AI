## 产品动机
每一年毕业季，毕业照总是最重要的环节，通过人脸识别与毕业照结合，可以成为您的校园回忆录。
## 产品目标 
通过人脸识别及时提取照片中人物的个人信息，如姓名、班级、专业、梦想职业、座右铭等信息，避免您的“脸盲症”。
